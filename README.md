# Perpustakaan

Halaman katalog buku statis yang mengambil data dari Google Sheets CSV.

## Deploy ke Vercel

1. Upload atau import folder ini ke Vercel:
   `/Users/foxie/Documents/Codex/2026-04-28/doctype-html-html-lang-id-head`
2. Pastikan file `index.html` berada di root project.
3. Framework Preset: `Other`.
4. Build Command: kosongkan.
5. Output Directory: kosongkan atau gunakan default.
6. Deploy.

Jika sebelumnya muncul `404: NOT_FOUND`, biasanya folder yang di-deploy tidak memiliki `index.html` di root, atau URL preview/deployment yang dibuka bukan deployment terbaru.
